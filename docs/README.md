# Flower Exhibition Robot — Documentation

Knowledge-base system (KBS) for the **Smart Flower Exhibition** assignment: a robot delivers bouquets from a central warehouse to pavilions on a grid, using **Experta** rules and **A\*** search.

## Quick start

```bash
pip install -r requirements.txt
python flower_robot_kbs.py
open index.html
```

## What this project achieves

1. **Rule-based expert system** — Experta `KnowledgeEngine` with facts and rules for moves, loads, unloads, constraints, and goals.
2. **Search tree** — Every legal successor state is generated by rules; the tree is recorded and printed.
3. **Optimal path** — A\* expands states in order of `f(n) = g(n) + h(n)` and finds minimum total cost.
4. **DFS comparison** — Same rule engine with depth-first frontier order (suboptimal but valid).
5. **Visualization** — `index.html` replays the optimal solution on the grid.
